# prd-drafts/ — proposals, isolated (catalog)

The **catalog** of proposals **not yet approved** — draft PRDs for systems that aren't built. A draft is
not a contract, and a ratified PRD in [`../prd/`](../prd/) may never cite one; that isolation is the whole
reason this folder exists, and `doc-lint` enforces it. This README is navigation: **the rules for drafts,
IDs, and graduation are in [`../guides/docs-prd.md`](../guides/docs-prd.md)** (see its *Drafts and
graduation* section).
